## 功能介绍

问卷星（wjx.cn）自动答题脚本，支持自动识别题型、批量填答和自动提交。

### 支持的题型

- 单选题 / 多选题
- 填空题（支持多个备选答案）
- 矩阵单选 / 矩阵多选
- 量表题（数字量表 + 星级评分）
- 下拉选择题

### 高级模式 - 概率答题

为每个选项设置被选中的概率，支持四种预设分布：

- **正态分布** — 中间选项概率高
- **左偏分布** — 低分选项概率高
- **右偏分布** — 高分选项概率高
- **均匀分布** — 所有选项等概率

也可手动精确调整每个选项的概率值。

### 其他特性

- 可拖拽悬浮窗，支持调整大小和位置记忆
- 自动提交模式，支持批量循环提交
- 模拟真实用户操作，处理人机验证
- 答案和设置自动保存到本地
- 快捷键 F3 显示/隐藏悬浮窗
- 自动解除页面右键和复制限制

### 使用方法

1. 安装脚本后打开任意问卷星问卷
2. 页面右侧出现悬浮窗，自动识别所有题目
3. 在悬浮窗中填写答案，点击「提交」
4. 开启「高级模式」可编辑每题的概率分布
5. 开启「自动提交」可批量循环填答
