## 问卷星自动答题脚本

一款功能强大的问卷星（wjx.cn）自动答题工具，支持**自定义概率比例**、**批量循环提交**，无需改代码，可视化操作界面，安装即用。

![脚本截图](https://raw.githubusercontent.com/PeanutSplash/Surveygen/main/src/assets/image/screenshot.png)

## 支持题型

- 单选题 / 多选题
- 填空题（可设置多个备选答案）
- 矩阵单选 / 矩阵多选
- 量表题（数字量表 + 星级评分）
- 下拉选择题

## 核心功能

### 自定义概率比例

为每个选项设置被选中的概率，四种预设分布一键切换：

- **正态分布** — 中间选项概率高，适合中立倾向
- **左偏分布** — 低分选项概率高
- **右偏分布** — 高分选项概率高
- **均匀分布** — 所有选项等概率

也可手动精确调整每个选项的概率值，概率实时可视化显示。

### 批量循环提交

开启自动提交后，脚本自动循环填答并提交问卷，支持：
- 自动填答所有题目
- 模拟真人操作，随机延迟
- 自动处理滑块验证
- 自动清除 Cookie 绕过设备限制
- 实时显示已提交份数

### 可视化操作界面

- 可拖拽悬浮窗，大小位置可调
- 所有题目自动识别，答案可视化编辑
- 概率分布颜色可视化
- 快捷键 F3 显示/隐藏
- 所有设置自动保存到本地

## 使用方法

1. 安装脚本后打开任意问卷星问卷页面
2. 页面右侧出现悬浮窗，自动识别所有题目
3. 在悬浮窗中设置答案，点击「提交」即可
4. 开启「高级模式」可编辑每题的概率分布
5. 开启「自动提交」可批量循环填答提交

## 其他特性

- 自动解除页面右键和复制限制
- vm 链接自动跳转到 vj 版本
- 答案和概率设置按问卷保存，下次打开自动恢复
